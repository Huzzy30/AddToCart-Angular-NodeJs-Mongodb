First we need to run the backend than we will run the angular,

For backend
1. open the directory NodeJs - Backend
2. then run the cmd execute it 'Nodemon app.js'
it will start the backend application at port 4300,

For Angular
1. open the directory Angular UI
2. then run the cmd execute it 'ng serve' 
it will start the fornt end application at port 4200
you can browse it http://localhost:4200
